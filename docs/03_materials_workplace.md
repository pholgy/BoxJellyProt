# วัสดุอุปกรณ์และสถานที่ดำเนินงาน (Materials and Workplace)

### Project: Jellyfish Toxin Drug Discovery using Molecular Docking

## 4. ข้อมูลที่ใช้ในการศึกษา (Data)

### โปรตีนพิษแมงกะพรุน (12 ชนิด)
- **Chironex fleckeri:** CfTX-1, CfTX-2, CfTX-A, CfTX-B
- **Nemopilema nomurai:** NnV-MLP, NnV-PLA2, NnV-CTX, NnV-Hemolysin
- **สายพันธุ์อื่นๆ:** Carybdea, Chrysaora, Pelagia, Cyanea

### สารยาที่ทดสอบ (25 ชนิด)
- **ฟลาโวนอยด์:** Silymarin, Quercetin, Kaempferol, Apigenin, Luteolin
- **สารยับยั้ง MMP:** EDTA, Marimastat, Batimastat, Doxycycline
- **ยาต้านอักเสบ:** Aspirin, Ibuprofen, Indomethacin
- **สารจากธรรมชาติ:** Curcumin, Resveratrol, EGCG
